# ProductManagementSystem

Steps to Run Project

1. Steps to Run Migration:


1.Change the userId and password based on your system connection string
2.Go to Pacakage Manager Console
3. Run add-migration InitialPMS1
4.Run update-database –verbose


2. Steps to Run Angular Project



1. Make sure you have Angular install on system
2.Run the ProductManagementAPI in VS instanc
3. Open terminal go to UI folder and run ngserve Command.
